01

¿Qué es Bootstrap 5?

🧰 ¿Qué es un Framework?

Un framework es un conjunto de herramientas, reglas y código ya escrito que nos da una base sólida para construir proyectos más rápido y de forma ordenada. En lugar de empezar desde cero, usamos lo que el framework ya nos provee.

🏠 Analogía: Construir sin framework es como fabricar cada ladrillo de tu casa desde cero. Un framework es como comprar los ladrillos ya hechos, estandarizados y listos para usar — vos solo te preocupás por construir.

📱 ¿Qué es Responsive (Diseño Responsivo)?

Un diseño responsivo es aquel que se adapta automáticamente al tamaño de la pantalla del dispositivo que lo muestra — celular, tablet, laptop o monitor — sin perder su usabilidad ni estética.

🎨 ¿Qué es un Framework CSS?

Un framework CSS es una colección de clases CSS predefinidas que podemos aplicar directamente en el HTML para agregar estilos sin escribir CSS desde cero. Bootstrap es el framework CSS más popular y usado del mundo.

Bootstrap 5 es el framework CSS más popular del mundo.

Creado por Twitter en 2011 y hoy mantenido como proyecto open source. La versión 5 eliminó la dependencia de jQuery y es completamente compatible con navegadores modernos.

Con Bootstrap podemos construir sitios responsive, modernos y consistentes escribiendo mucho menos CSS — simplemente agregamos clases a nuestro HTML.

✅ Gratuito ✅ Open Source ✅ Responsive ✅ Sin jQuery en v5
02

Pilares de Bootstrap

Bootstrap se organiza en tres grandes pilares que lo hacen tan poderoso y completo:

📐

Grid System

Sistema de rejilla de 12 columnas que permite crear layouts flexibles y responsive. Es la base estructural de cualquier página con Bootstrap.

🧩

Componentes

Bloques de HTML y CSS reutilizables ya diseñados: navbars, cards, botones, formularios, modales, alertas y más de 30 componentes listos para usar.

🎨

Iconos

Librería oficial de más de 2,000 iconos SVG gratuitos, disponibles vía CDN. Se integran con una simple clase en cualquier elemento HTML.

03

Cómo Incluir Bootstrap en tu Proyecto

La forma más rápida y sencilla es usando el CDN (Content Delivery Network) — una red de servidores que nos entrega los archivos de Bootstrap directamente desde internet, sin necesidad de descargar nada.

🌐 ¿De dónde se obtienen los enlaces CDN?

Los enlaces oficiales siempre se consiguen desde la documentación oficial de Bootstrap en:
👉 getbootstrap.com/docs/5.3/getting-started/download/

Dentro de esa página, buscá la sección "CDN via jsDelivr" — ahí están los dos enlaces (CSS y JS) actualizados con la versión más reciente.

1

Crear la estructura HTML base

Abrí VS Code, creá tu archivo index.html y generá la estructura con el atajo ! + Tab.

2

Pegar el CSS de Bootstrap en el <head>

Copiá el siguiente enlace y pegalo antes de cerrar la etiqueta </head>:

index.html — dentro del <head> 
<!-- CSS de Bootstrap 5 — va en el <head> -->
<link href="https://cdn.jsdelivr.net/npm/[email protected]/dist/css/bootstrap.min.css"
      rel="stylesheet"
      integrity="sha384-sRIl4kxILFvY47J16cr9ZwB07vP4J8+LH7qKQnuqkuIAvNWLzeN8tE5YBujZqJLB"
      crossorigin="anonymous">
3

Pegar el JS de Bootstrap también en el <head>

Copiá el siguiente script y pegalo dentro del <head>, justo debajo del enlace CSS. Ambos enlaces van en el <head>.

index.html — dentro del <head> 
<!-- JS de Bootstrap 5 — también va en el <head> -->
<script src="https://cdn.jsdelivr.net/npm/[email protected]/dist/js/bootstrap.bundle.min.js"
        integrity="sha384-FKyoEForCGlyvwx9Hj09JcYn3nv7wiPVlz7YYwJrWVcXK/BmnVDxM+D2scQbITxI"
        crossorigin="anonymous"></script>

El Sistema Grid de Bootstrap

📐 ¿Qué es la Grid?

La Grid (rejilla o cuadriculado) es el sistema que Bootstrap usa para organizar el contenido en filas y columnas. Funciona como una tabla invisible que divide la pantalla en 12 columnas de igual tamaño, permitiéndonos controlar cuánto espacio ocupa cada elemento.

Breakpoints

📏 ¿Qué es un Breakpoint?

Un breakpoint es un punto de quiebre — un ancho de pantalla específico en el que el diseño cambia para adaptarse mejor al dispositivo. Bootstrap tiene 6 breakpoints predefinidos que cubren desde celulares pequeños hasta monitores grandes.

🏠 Analogía: Es como el agua que toma la forma del recipiente — en un vaso pequeño (celular) el contenido se apila verticalmente, en un recipiente grande (monitor) se distribuye en columnas.
Nombre Sufijo en clase Ancho mínimo Dispositivo típico
Extra smallninguno (.col-)< 576pxCelulares pequeños
Small.col-sm-≥ 576pxCelulares grandes
Medium.col-md-≥ 768pxTablets
Large.col-lg-≥ 992pxLaptops
Extra large.col-xl-≥ 1200pxDesktops
Extra extra large.col-xxl-≥ 1400pxMonitores grandes

Contenedores de Bootstrap

El contenedor es el elemento padre que envuelve las filas y columnas. Bootstrap ofrece dos tipos principales y contenedores responsivos para cada breakpoint.

.container-fluid 100% del ancho siempre
.container Ancho máximo según breakpoint
.container-lg 100% hasta lg, luego fijo
Clase <576px ≥576px ≥768px ≥992px ≥1200px ≥1400px
.container100%540px720px960px1140px1320px
.container-sm100%540px720px960px1140px1320px
.container-md100%100%720px960px1140px1320px
.container-lg100%100%100%960px1140px1320px
.container-xl100%100%100%100%1140px1320px
.container-xxl100%100%100%100%100%1320px
.container-fluid100%100%100%100%100%100%
Uso de contenedores 
<!-- Ancho máximo centrado -->
<div class="container">
  Contenido centrado
</div>

<!-- Ancho completo siempre -->
<div class="container-fluid">
  Ancho total
</div>

<!-- 100% hasta lg, luego fijo -->
<div class="container-lg">
  Responsive hasta lg
</div>
📌 ¿Cuándo usar cada uno?
  • 🔹 .container — Para la mayoría de páginas. Centra y limita el ancho.
  • 🔹 .container-fluid — Para secciones hero, headers o cuando querés que ocupe toda la pantalla.
  • 🔹 .container-{bp} — Cuando querés control fino sobre en qué tamaño de pantalla empieza a limitarse.

Componentes de Bootstrap

🧩 ¿Qué es un Componente?

Un componente es un bloque de HTML reutilizable ya diseñado y estilizado por Bootstrap. Solo copiamos el código del componente, lo pegamos en nuestro HTML y funciona de inmediato — sin escribir CSS ni JS adicional.

Bootstrap incluye más de 30 componentes. Los más usados en el curso serán:

🧭 Navbar

Barra de navegación responsive con menú hamburguesa

🃏 Cards

Tarjetas de contenido con imagen, título y texto

🔘 Buttons

Botones con múltiples estilos y tamaños

📝 Forms

Formularios estilizados con validación visual

🪟 Modal

Ventanas emergentes con contenido dinámico

🔔 Alerts

Mensajes de éxito, error, advertencia e información

Bootstrap Icons

Bootstrap Icons es la librería oficial de iconos de Bootstrap, con más de 2,000 iconos SVG gratuitos y de código abierto.

Están disponibles vía CDN — simplemente agregamos el enlace en el <head> y usamos una clase <i> en nuestro HTML.

Paso 1 — Agregar en el <head> 
<link rel="stylesheet"
      href="https://cdn.jsdelivr.net/npm/[email protected]/font/bootstrap-icons.min.css">
Paso 2 — Usar el icono en el HTML 
<!-- Formato: bi bi-[nombre-del-icono] -->
<i class="bi bi-house"></i>          <!-- 🏠 Casa -->
<i class="bi bi-envelope"></i>       <!-- ✉️ Correo -->
<i class="bi bi-person-circle"></i>  <!-- 👤 Usuario -->
<i class="bi bi-github"></i>         <!-- 🐙 GitHub -->

<!-- Con tamaño y color mediante clases utilitarias -->
<i class="bi bi-star-fill fs-3 text-warning"></i>
<i class="bi bi-check-circle fs-2 text-success"></i>

<!-- Dentro de un botón -->
<button class="btn btn-primary">
  <i class="bi bi-send"></i> Enviar
</button>

Crear el repositorio del ejercicio

06
